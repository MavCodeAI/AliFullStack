# Scaffold Vue

A basic Vue 3 + Vite setup with TypeScript and Tailwind CSS.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:8080`

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
